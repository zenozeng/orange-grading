# orange-grading

问题描述：
输入 100g 到 600g 的橙子（正态分布），
分为多个级别比如 100-110,111-120等（分组上限是给定的，16）。

给定橙子的重量，比如 14公斤，
估计一个线性表达式，
从各个组里给出一个固定组合（线性表达式），
计算误差。

求一种最佳的分组方法，使得其误差最小。
